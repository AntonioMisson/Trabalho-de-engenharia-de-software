Sobre o Projeto

Este sistema foi desenvolvido como parte do trabalho da disciplina de Engenharia de Software. O objetivo é demonstrar o desenvolvimento de um sistema web completo com funcionalidades essenciais, organização de código, testes automatizados e persistência de dados.

Funcionalidades

  • Cadastro e gerenciamento de usuários
  • Persistência de dados em arquivos JSON
  • Testes automatizados para garantir a qualidade
